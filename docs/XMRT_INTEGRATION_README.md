# XMRT-Eliza Integration

This repository contains the XMRT-DAO ecosystem integration for Eliza.

## Features
- Multi-agent coordination
- Real-time communication with XMRT ecosystem
- Knowledge sharing and task orchestration
- Council reporting and governance participation

## Quick Start
1. Configure environment variables
2. Install dependencies: `npm install`
3. Run with: `npm run dev -- --character=characters/xmrt-eliza.character.json`
